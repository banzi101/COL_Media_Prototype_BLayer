# Introduction 
an api that uses api authentication and jwt tokens to saves media information on a mongoDB. 

# Getting Started
Blayer setup:
1. in the terminal run the command:
	`npm install`
	
2. create a .env file containing:
	`PORT=___`
	`API_KEY=___`
	`JWT_SECRET=___`
    `REFRESH_TOKEN_SECRET=___`

# Build and Test
1. in the terminal run the command:
    `npm start`
    OR
    for testing:
    `npm run dev`
